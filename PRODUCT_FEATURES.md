## 1. 平台簡介

**Learning Platform** 是一個線上家教預約平台，學生可以自由瀏覽並預約各科目老師的一對一課程，老師則能在平台上展示個人資歷、設定排程、管理預約並獲得收入。

平台支援三種使用者角色，覆蓋完整的學習與教學生命週期：

| 角色 | 說明 |
|------|------|
| 🎓 **學生（Student）** | 瀏覽課程、預約時段、線上上課、課後評價 |
| 👨‍🏫 **老師（Tutor）** | 開設課程、管理排程、即時授課、查看收入 |
| 🛡️ **管理員（Admin）** | 審核老師資格、監控平台數據與營收 |

### 技術棧

`Spring Boot 3` · `Spring Security + JWT` · `Spring WebSocket (STOMP)` · `JPA / Hibernate`
`Vanilla JS` · `Axios` · `Bootstrap 5` · `Chart.js` · `WebRTC` · `ECPay`

## 3. 帳號系統

學生可直接在平台上免費註冊，擁有帳號後即可瀏覽課程、預約老師。若希望成為老師，可在帳號內提交申請，經管理員審核通過後即可開始接課。

### 功能重點

- 📝 **快速註冊**：填寫姓名、Email 與密碼即可完成學生帳號建立
- 🔑 **安全登入**：採用 JWT（JSON Web Token）進行身份驗證，無需 Session
- 👤 **個人資料管理**：可隨時更新姓名、生日，以及修改登入密碼
- 🔀 **角色升級**：學生帳號可申請成為老師，審核通過後即可使用老師功能
- 🔒 **自動登出保護**：Token 逾期或未授權時，系統自動登出並引導重新登入

## 4. 課程探索與搜尋

學生可在探索頁面瀏覽平台所有課程，透過多維度篩選快速找到符合需求的老師與科目。

### 功能重點

- 🔍 **多條件篩選**：依科目類別、上課時間（週幾 / 幾點）、價格區間進行組合篩選
- 🃏 **老師卡片瀏覽**：首頁展示老師卡片走馬燈，支援拖曳滑動
- 📅 **翻牌排程預覽**：每張課程卡片可翻面查看老師本週可用時段，一目瞭然
- 📄 **分頁載入**：課程列表支援分頁，瀏覽更流暢
- ⭐ **體驗課標示**：符合資格的學生可直接在探索頁辨識並預約體驗課

## 5. 老師個人頁

每位老師都有專屬的個人頁，學生可以深入了解老師的背景、教學風格與排程，再決定是否預約。

### 功能重點

- 🪪 **完整個人檔案**：頭像、頭銜、自我介紹、學歷與教學經歷
- 🎥 **介紹影片**：老師可上傳最多 2 支介紹影片，讓學生感受教學風格
- 📜 **證照與證書**：可展示教師資格或相關認證文件
- 📊 **課程評分**：顯示各課程的平均星級評分與學生留言
- 📆 **即時排程檢視**：顯示老師本週的可預約時段（已預約時段自動隱藏）
- 🛒 **直接預約**：在個人頁可直接選課並前往訂課流程

## 6. 訂課與付款流程

學生選定老師與課程後，可在訂課頁面靈活選取時段、選擇套餐方案，確認後以點數完成付款。

### 訂課步驟

```
① 選擇課程  →  ② 選取時段（最多 4 週內）
→  ③ 選擇堂數套餐  →  ④ 確認點數餘額  →  ⑤ 完成購買
```

### 套餐方案

| 方案 | 堂數 | 折扣 |
|------|------|------|
| 單堂 | 1 堂 | 原價 |
| 五堂包 | 5 堂 | 95 折（9.5 折） |
| 十堂包 | 10 堂 | 9 折 |
| 體驗課 | 1 堂 | 固定優惠價 200 點 |

### 功能重點

- 🗓️ **四週日曆選時**：可在未來 4 週內自由選取可用時段
- ⏰ **提前 24 小時限制**：訂課需於上課前 24 小時前完成
- 🚫 **防重複預約**：系統自動偵測老師與學生的時段衝突，杜絕重複訂課
- 💰 **點數扣除確認**：購買前顯示總金額與帳戶餘額，點數不足時引導前往儲值
- 📧 **預約確認信**：購買成功後自動發送 Email 通知老師

## 7. 點數錢包與儲值

平台採用「點數」制度作為交易媒介，學生以新台幣儲值後取得點數，再用點數支付課程費用；老師的授課收入也以點數形式累積至帳戶。

### 儲值方案（ECPay 金流）

| 儲值金額 | 加贈點數 | 實得點數 |
|----------|----------|----------|
| NT$1,500 | +200 | 1,700 |
| NT$3,000 | +600 | 3,600 |
| NT$5,000 | +1,200 | 6,200 |

### 點數流向

| 類型 | 說明 | 發生時機 |
|------|------|----------|
| ➕ 儲值 | 付款成功後點數自動入帳 | ECPay 付款完成 |
| ➖ 購課 | 依套餐折扣後的總金額扣除 | 預約送出時 |
| ➕ 授課收入 | 課程結束後系統自動撥款 | 每小時自動排程 |
| ➕ 退款 | 取消預約或整筆退款 | 學生申請退款 |

### 功能重點

- 🏦 **ECPay 金流整合**：支援台灣主流支付方式
- 🎁 **儲值加贈獎勵**：金額越高，加贈比例越大
- 📋 **交易明細記錄**：每筆儲值、扣款、退款均記錄在案，可查詢完整歷史
- 🔐 **防重複入帳**：系統以唯一訂單號防止網路重試造成重複扣/入帳

## 8. 學生課程管理

學生可在個人後台全覽所有購買的課程與預約記錄，並進行取消或退款操作。

### 功能重點

- 📚 **課程包管理**：查看各堂數包的已用堂數、剩餘堂數與購買價格
- 🗂️ **分類顯示**：課程依「即將到來」、「已完成」、「已取消」分頁呈現
- ❌ **取消預約**：
  - 距離上課時間 **超過 12 小時**：取消後全額退款至點數帳戶
  - 距離上課時間 **不足 12 小時**：不予退款
- 💸 **整筆退款**：可將整個課程包（含剩餘未上課程）一次退款
- ⭐ **課後評價**：課程完成後可對老師與課程留下星級評分及文字評論
- 📅 **今日 / 未來課表**：儀表板顯示今天的課與未來 4 週的預約

## 9. 老師課程管理

老師擁有完整的後台工具，可自主管理課程、排程與收入，同時追蹤教學狀況。

### 功能重點

- 📖 **課程開設與編輯**：新增課程時設定名稱、科目、課程簡介與每堂價格
- 🗓️ **週排程設定**：以 7 × 13（週一至週日 × 09:00–21:00）格狀介面設定可用時段
  - 支援單格切換或批次更新多個時段
- 👁️ **預約總覽**：查看所有學生的預約紀錄，包含學生姓名、課程、日期與狀態
- 📈 **統計儀表板**：顯示本週授課堂數、本月收入、平均評分與今日課程數
- 💹 **收入分析圖表**：以長條圖呈現每日 / 每月收入趨勢
- 📝 **課後回饋**：對學生的課程表現填寫評語，供學生參考

## 10. 即時聊天

學生與老師可透過聊天功能進行課前確認、課後追蹤，支援文字與多媒體內容傳送。

### 功能重點

- 💬 **對話列表**：以對象（老師 / 學生）分組顯示所有對話，含最後訊息預覽
- 🔎 **對話搜尋**：可快速搜尋特定對話
- 📎 **多媒體傳送**：支援文字、圖片、影片、語音訊息、貼圖與一般檔案
- 🔗 **連結預覽**：貼入網址時自動產生預覽卡片
- 📥 **安全下載**：媒體檔案下載需驗證身份，保護隱私
- 🕐 **時間格式化**：今天 / 昨天 / 本週 / 完整日期的智慧時間顯示

## 11. 視訊教室

每堂課程都有專屬的視訊教室，老師與學生可在預約時段內進行點對點即時視訊授課，並內建課中聊天功能。

### 功能重點

- 📹 **即時視訊**：採用 WebRTC 點對點技術，無需安裝任何外部軟體
- 🖥️ **螢幕分享**：老師可分享螢幕進行講解或示範
- 🎤 **麥克風 / 鏡頭控制**：可隨時靜音或關閉鏡頭
- 💬 **課中即時聊天**：教室內建聊天欄，支援文字與媒體傳送，訊息自動儲存
- 🚪 **進入前測試**：加入教室前可預先測試鏡頭與麥克風是否正常
- 👥 **參與者管理**：顯示目前在線的參與者名單，記錄進出教室事件

## 12. 管理後台

管理員擁有獨立後台，可全面監控平台運作狀況，並負責審核新加入的老師。

### 功能重點

- 📊 **平台數據儀表板**：
  - 總學生人數、老師人數、課程類型數
  - 本月新增學員數
  - 平台每日 / 每月營收
  - 最受歡迎課程 Top 5
- 👨‍🏫 **老師審核管理**：
  - 分頁檢視待審核（Pending）、已通過（Qualified）、已停權（Suspended）的老師
  - 查看老師申請資料、學歷與證照
  - 一鍵通過或停權老師帳號
- 🔢 **老師統計摘要**：快速掌握各狀態老師的數量分布

## 13. 核心業務流程

### 學生完整使用旅程

```
[帳號建立]
  └─ 註冊帳號（學生角色）→ 登入
        │
        ▼
[探索課程]
  └─ 探索頁篩選課程 → 查看老師個人頁 → 閱讀評價與排程
        │
        ▼
[儲值點數]（如餘額不足）
  └─ 選擇儲值方案 → ECPay 付款 → 點數入帳（含加贈）
        │
        ▼
[預約課程]
  └─ 選擇時段 → 選擇套餐 → 確認點數 → 完成購買
        │             └─ 老師收到預約確認 Email
        ▼
[上課]
  └─ 進入視訊教室 → 即時視訊授課 → 課中聊天
        │
        ▼
[課後]
  └─ 留下評價 → 系統自動退費（若取消）或撥款給老師
```

---

### 老師加入流程

```
[申請]
  └─ 學生帳號 → 提交老師申請
        │
        ▼
[審核]
  └─ 管理員在後台審查申請資料
        ├─ 通過（status=2）→ 老師資格啟用
        └─ 停權（status=3）→ 帳號暫停
        │
        ▼
[設定]
  └─ 填寫個人檔案（頭像、自介、學歷、影片）
     → 開設課程（名稱、科目、價格）
     → 設定週排程（可接課時段）
        │
        ▼
[接課與收款]
  └─ 學生預約 → 進行視訊授課
     → 課程結束後系統自動計算並撥款至點數帳戶
        ├─ 體驗課：收取 100% 課程費用
        └─ 正式課：收取 80% 課程費用（平台抽成 20%）
```

## 14. 平台特色亮點

### 🎁 體驗課機制
每位學生可享有 **一次限定的體驗課**，以優惠固定價格（200 點）體驗任一老師的課程，降低學習入門門檻。

---

### ⏱️ 12 小時退款保護
訂課後若需取消，距離上課時間 **超過 12 小時** 可獲得全額退款；系統規則透明，保障學生權益。

---

### 🤖 課後自動撥款
老師無需手動申請提領。課程結束後，系統每小時自動偵測已完成的課程，將授課費用直接撥入老師的點數帳戶，流程全自動化。

---

### 🎁 儲值加贈獎勵
儲值金額越高、加贈比例越大（最高加贈 24%），鼓勵學習投入，同時提供學生更高的課程學習彈性。

---

### 📡 WebRTC 點對點視訊
視訊教室採用 **WebRTC** 技術實現瀏覽器原生點對點連線，學生與老師無需安裝任何額外軟體，開啟瀏覽器即可上課。

---

### 🔐 全站 JWT 身份驗證
後端 API 與 WebSocket 連線均採用 **JWT Bearer Token** 進行身份驗證，確保每一個請求都有明確的身份授權，有效防止未授權存取。
